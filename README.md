## 📌 Proyectos destacados

### 📊 Olist E-commerce Power BI Dashboard  
Dashboard de Business Intelligence para analizar ventas, clientes, productos, logística, satisfacción y métodos de pago en un negocio e-commerce.  
**Herramientas:** Power BI, DAX, Python, Pandas, KPIs, modelado de datos.  
🔗 [Ver proyecto](https://github.com/mmunitavidal/olist-ecommerce-powerbi-dashboard)

### 🧪 Marketing Analytics & A/B Testing  
Análisis de funnel de conversión y evaluación de experimento A/B para apoyar decisiones de producto y marketing.  
**Herramientas:** Python, SQL, Tableau, SciPy, pruebas estadísticas.  
🔗 [Ver proyecto](https://github.com/mmunitavidal/Marketing-AB-Testing-Customer-Analytics)

### 🚀 Marketing Mix Modeling & Budget Optimization  
Modelo predictivo para analizar la relación entre inversión en medios y ventas, apoyando decisiones de optimización presupuestaria.  
**Herramientas:** Python, Scikit-learn, Machine Learning, Ridge Regression, KPIs de marketing.  
🔗 [Ver proyecto](https://github.com/mmunitavidal/marketing-mix-modeling-budget-optimization)

### 🎧 Spotify Streaming Audit  
Auditoría de catálogo musical para priorizar canciones con potencial premium usando SQL, Machine Learning y visualización.  
**Herramientas:** Python, SQL, DuckDB, K-Means, Random Forest, Tableau.  
🔗 [Ver proyecto](https://github.com/mmunitavidal/spotify-streaming-audit)

### 📈 Marketing Funnel Optimization & Growth Analysis  
Análisis de embudo digital para detectar puntos de fuga, oportunidades de conversión y mejoras de segmentación.  
**Herramientas:** Python, Pandas, Matplotlib, Seaborn, KPIs de conversión.  
🔗 [Ver proyecto](https://github.com/mmunitavidal/marketing-funnel-optimization-analysis)

### 💰 Customer Revenue & Plan Performance Analysis  
Análisis de ingresos y comportamiento de clientes para evaluar rentabilidad, desempeño de planes, pricing y oportunidades de segmentación.  
**Herramientas:** Python, Pandas, NumPy, Matplotlib, pruebas estadísticas.  
🔗 [Ver proyecto](https://github.com/mmunitavidal/customer-revenue-analysis-megaline)
